Two Main Strategies:

üîÑ Iteration

Repeating a process using loops

                    

function factorialIterative(n) {
    let result = 1;
    for (let i = 1; i <= n; i++) {
        result = result * i;
    }
    return result;
}

// factorial(5) = 1*2*3*4*5 = 120

When to use: When you need to process items one by one, or repeat until a condition is met.

üîÅ Recursion

A function calling itself with simpler inputs

                    

function factorialRecursive(n) {
    if (n <= 1) {
        return 1;  // Base case
    }
    return n * factorialRecursive(n - 1);
}

// factorial(5) = 5 * factorial(4)
//              = 5 * 4 * factorial(3)...

When to use: When the problem can be broken into smaller, similar subproblems.
