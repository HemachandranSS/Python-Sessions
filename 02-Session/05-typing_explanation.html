Static vs Dynamic Typing

Static Typing

# Python with type hints (static-like) def add_numbers(a: int, b: int) -> int: return a + b # Variable types are declared name: str = "Alice" age: int = 25 is_student: bool = True

Variables with Fixed Types:

name: str
age: int
is_student: bool

Dynamic Typing

# Python (dynamically typed) def add_anything(a, b): return a + b # Variables can change types variable = "Hello" # string variable = 42 # now it's a number variable = [1, 2, 3] # now it's a list

Variable that Changes:

variable: "Hello"

Key Differences:

Static Typing:

  • Types are checked at compile time
  • Variables have fixed types once declared
  • Catches type errors before runtime
  • Examples: Java, C++, TypeScript, Python with type hints

Dynamic Typing:

  • Types are checked at runtime
  • Variables can change types during execution
  • More flexible but can have runtime errors
  • Examples: Python, JavaScript, Ruby, PHP
