üîçWhat is Boolean?

The Boolean data type in Python represents logical values with only two possible states: True and False. Named after mathematician George Boole, this data type is fundamental to logic and decision-making in programming.

True

Internally: 1

False

Internally: 0

Booleans can be created directly or as results of comparisons and logical operations:

üí° Truthy and Falsy Values

Falsy values: False, None, 0, 0.0, '', [], {}, set()
Truthy values: Everything else including True, non-zero numbers, non-empty strings, non-empty collections

üöÄ Key Points to Remember

  • Boolean values are case-sensitive: True and False (not true or false)
  • Python internally treats True as 1 and False as 0
  • Any non-zero number, non-empty string, or non-empty collection is True
  • Boolean operations follow short-circuit evaluation
  • Booleans can be used in arithmetic operations
