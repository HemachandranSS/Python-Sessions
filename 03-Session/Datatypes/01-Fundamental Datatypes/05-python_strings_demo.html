üìö What is a String?

A String is a sequence of characters enclosed within quotes. In Python, strings are immutable, meaning once created, they cannot be changed.

Single Quotes

                    
s1 = 'Hema'

Double Quotes

                        
s1 = "Hema"

Triple Quotes (Multi-line)

                    
s1 = '''Hema
Software
Engineer'''

üéØ String Indexing

Python strings follow zero-based indexing. Each character has both positive (left to right) and negative (right to left) indices.

üîß String Operations

String Repetition (*)

>> s = "Hema"
>>> s * 3
'HemaHemaHema'

String Length (len)

>> s = "Hema"
>>> len(s)
4

Character Access

>> s = "Hema"
>>> s[0]
'H'
>>> s[-1]
'a'

String Concatenation (+)

>> s1 = "Hema"
>>> s2 = " rocks!"
>>> s1 + s2
'Hema rocks!'

üí° Key Points to Remember

  • ‚úÖ Strings are immutable - operations create new strings
  • ‚úÖ Use triple quotes for multi-line strings
  • ‚úÖ Negative indices count from the end (-1 is last character)
  • ‚úÖ Slicing syntax: [start:end:step] (end is exclusive)
  • ‚úÖ IndexError occurs when accessing invalid indices
