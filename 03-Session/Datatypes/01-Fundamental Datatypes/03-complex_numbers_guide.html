🧮 Python Complex Numbers

Explore the fascinating world of complex numbers in Python - from basic syntax to real-world applications

📐 Complex Number Structure

a + bj

Real Part (a)

Integer or Float

Can be: decimal, binary, octal, hex

Imaginary Part (b)

Coefficient of j

Must be in decimal form

Key Point: j² = -1, where j = √(-1)

💻 Valid Examples

            

# Valid complex numbers
a = 3 + 5j
b = 10 + 5.5j
c = 0.5 + 0.1j
d = 0B11 + 5j  # Binary real part (3 + 5j)

print(a)  # Output: (3+5j)
print(type(a))  # Output: <class 'complex'>

❌ Invalid Syntax

            

# This will cause SyntaxError
a = 3 + 0B11j  # Invalid: imaginary part must be decimal
# SyntaxError: invalid syntax

# Correct way:
a = 3 + 11j  # Use decimal form for imaginary part
