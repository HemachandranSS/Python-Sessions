NUMERIC
1
Integer (int)
Whole numbers without decimal points. Can be positive, negative, or zero. Python 3 integers have unlimited precision!
x = 42 y = -17 z = 0 big_num = 123456789012345678901234567890
NUMERIC
1.0
Float
Numbers with decimal points. Represented in double precision floating-point format.
pi = 3.14159 scientific = 1.23e-4 # Scientific notation infinity = float('inf')
NUMERIC
i
Complex
Numbers with real and imaginary parts. Written as (real + imaginary*j).
z1 = 3 + 4j z2 = complex(2, -3) # 2 - 3j real_part = z1.real # 3.0 imag_part = z1.imag # 4.0
BOOLEAN
T/F
Boolean (bool)
Represents truth values: True or False. Subclass of int where True=1 and False=0.
is_python_awesome = True is_boring = False result = True + True # 2 (because True = 1)
SEQUENCE
"A"
String (str)
Immutable sequence of Unicode characters. Can be created with single, double, or triple quotes.
name = "Python" multiline = """This is a multiline string""" f_string = f"Hello, {name}!"
Notes
üìí
Fundamental Datatypes in Python
In Python, int, float, complex, bool, str data types are considered as Fundamental Data types
SEQUENCE
[]
List
Mutable, ordered collection of items. Can contain different data types and supports indexing and slicing.
fruits = ["apple", "banana", "cherry"] mixed = [1, "hello", 3.14, True] nested = [[1, 2], [3, 4]]
SEQUENCE
()
Tuple
Immutable, ordered collection of items. Often used for coordinates, database records, or function returns.
coordinates = (3, 4) colors = ("red", "green", "blue") single = (42,) # Note the comma!
SEQUENCE
R
Range
Immutable sequence of numbers, commonly used for loops. Memory-efficient for large ranges.
numbers = range(5) # 0, 1, 2, 3, 4 evens = range(0, 10, 2) # 0, 2, 4, 6, 8 countdown = range(10, 0, -1)
SET
{}
Set
Mutable, unordered collection of unique items. Great for removing duplicates and set operations.
unique_numbers = {1, 2, 3, 4, 5} fruits = set(["apple", "banana", "apple"]) # Result: {"apple", "banana"}
SET
‚ùÑÔ∏è
Frozenset
Immutable version of set. Can be used as dictionary keys or elements in other sets.
immutable_set = frozenset([1, 2, 3, 4]) nested_sets = {frozenset([1, 2]), frozenset([3, 4])}
MAPPING
{:}
Dictionary (dict)
Mutable, unordered collection of key-value pairs. Keys must be immutable and unique.
person = {"name": "Alice", "age": 30} nested = {"coords": {"x": 10, "y": 20}} mixed_keys = {1: "one", "two": 2}
BINARY
b''
Bytes
Immutable sequence of bytes (0-255). Used for binary data like images, files, or network protocols.
data = b"Hello" encoded = "Hello".encode('utf-8') hex_bytes = bytes.fromhex('48656c6c6f')
BINARY
[]b
Bytearray
Mutable sequence of bytes. Like bytes, but you can modify individual elements.
data = bytearray(b"Hello") data[0] = ord('h') # Change 'H' to 'h' # Result: bytearray(b'hello')
NONE
‚àÖ
NoneType (None)
Represents the absence of a value. Often used as a default value or to indicate "no data".
result = None def no_return(): pass # Returns None implicitly value = no_return() # None

üõ†Ô∏è Essential Built-in Functions

Python provides several built-in functions to work with data types and objects:

type()

Returns the type of an object

type(42) # <class 'int'>

id()

Returns unique identifier (memory address) of an object

id("hello") # 140712234567408

isinstance()

Checks if an object is an instance of a specific type

isinstance(42, int) # True

len()

Returns the length of a sequence or collection

len([1, 2, 3]) # 3
