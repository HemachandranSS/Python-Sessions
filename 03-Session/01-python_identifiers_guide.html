Python Identifiers

Master the art of naming in Python programming

🐍What is a Python Identifier?

A Python Identifier is a name used to identify variables, functions, classes, modules, or other objects in a Python program. Think of it as a label that helps you reference different elements in your code.

# Examples of identifiers in action
variable_name = 10 # Variable identifier
def function_name(): # Function identifier
    pass
class ClassName: # Class identifier
    pass

📜The 7 Golden Rules of Python Identifiers

1

Allowed Characters Only

Use only alphabets (A-Z, a-z), digits (0-9), and underscore (_)

valid_name ✓
name123 ✓
ca$h ✗ // $ not allowed
2

No Leading Digits

Identifiers cannot start with a digit (0-9)

123total ✗
total123 ✓
_123total ✓
3

Case Sensitive

Python treats uppercase and lowercase as different

total = 10
TOTAL = 999
# These are different variables!
4

No Reserved Words

Cannot use Python keywords as identifiers

def = 10 ✗
if = "hello" ✗
def_value = 10 ✓
5

No Length Limit

Identifiers can be any length (but keep them reasonable!)

x ✓
very_long_variable_name ✓
# But avoid extremely long names
6

No Dollar Sign ($)

Unlike some languages, Python doesn't allow $ in identifiers

user$ ✗
$price ✗
user_account ✓
7

Underscore Conventions

Underscores have special meanings in Python

_private # Private
__very_private # Very private
__magic__ # Magic method

🌟 Special Underscore Conventions

Single Underscore (_name)

Indicates a private identifier (by convention)

_internal_var = 42

Double Underscore (__name)

Indicates a strongly private identifier

__secret_data = "hidden"

Magic Methods (__name__)

Language-defined special methods

__init__, __str__, __add__
